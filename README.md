# RentHub  
A Django-based rental management system for cars, bikes, and other vehicles.

## Overview
RentHub is a full-featured web application built with Django that allows users to:
- Browse available vehicles
- View detailed product pages
- View rentals list
- Add new products (admin)
- Upload vehicle images
- Manage categories and subcategories

This project is currently under development and will be expanded with more features.

---

## Features
- Vehicle listing (cars, bikes, etc.)
- Category & brand-based filtering
- User-friendly interface
- Admin panel to manage products
- Image uploads for logos & product photos
- Django ORM models & migrations
- Clean and structured folder system

---

## Tech Stack
- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (development)
- **Version Control:** Git & GitHub
